# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by sending an email to the maintainers or create a private GitHub Security Advisory.

Do NOT create a public issue for security vulnerabilities.

## Scope

This policy covers the HeadControl web application and its dependencies.

## Response Timeline

- **Initial Response:** 24-48 hours
- **Assessment:** Within 1 week
- **Fix:** As soon as possible based on severity
